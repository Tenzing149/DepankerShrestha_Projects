Image Filter Editor

This is a basic photo editor that consists of 5 filters that users can use and change the intensity of the filter. Filters are Gray Image, Black and White, Pencil Sketch, Blur Effect, and Invert. Python3 and its following libraries were used:

  • streamlit
  • numpy
  • pillow
  • opencv
